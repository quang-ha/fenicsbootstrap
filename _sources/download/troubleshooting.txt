###############
Troubleshooting
###############

If for some reason you were unable to obtain, build, or install a FEniCS
component, please let us know by sending a message to the `FEniCS
support mailing list
<http://fenicsproject.org/mailman/listinfo/fenics-support>`_.

If you have suggestions for improving the information found on this page
with respect to the installation procedure of FEniCS, you are more than
welcome to register an issue on `FEniCS Web
<https://bitbucket.org/fenics-project/fenics-web>`_.
